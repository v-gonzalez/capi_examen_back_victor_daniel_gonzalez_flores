# capi_examen_back_victor_daniel_gonzalez_flores
Examen de prueba para empresa CAPI


- Hacer copia del archivo .env.example con nombre .env
- Modificar en el archivo .env los valores de:
    DB_DATABASE=laravel
    DB_USERNAME=root
    DB_PASSWORD=
- Ejecutar el comando
    composer install
    php artisan migrate
    php artisan db:seed
